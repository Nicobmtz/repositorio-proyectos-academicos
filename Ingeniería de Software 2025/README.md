# Proyecto: Sistema de Asistencia al viajero

Este proyecto consiste en modelar un sistema de asistencia para turistas que visitan una nueva ciudad.
Para ello, se requiere realizar un análisis de los requerimientos del sistema y representarlo utilizando el lenguaje UML, mediante un modelo conceptual de clases, un diagrama de casos de uso y un diagrama de arquitectura de software.  
Asimismo, se solicita realizar un maquetado de una de las funciones principales del sistema.  
En esta carpeta se incluyen los archivos en los que participé activamente.  
Se omite el maquetado, ya que actualmente no cuento con el conocimiento necesario para realizar un trabajo similar.  
El archivo que acompaña al diagrama de arquitectura estará disponible próximamente.

---

## Objetivo del proyecto

- Realizar un modelo conceptual del dominio del problema.
- Identificar y detallar en un diagrama los distintos casos de uso del sistema.
- Diagramar el sistema utilizando alguna vista de arquitectura a elección.

## Aprendizaje y evolución del proyecto

Durante el desarrollo de este trabajo, el modelo conceptual atravesó varias revisiones y finalmente una reestructuración completa.
En las primeras versiones aún estaba afianzando los conceptos de análisis y modelado UML, lo que llevó a reformular gran parte del diagrama una vez comprendido con mayor profundidad el dominio del sistema.
Este proceso me permitió mejorar mi criterio para identificar entidades, relaciones y responsabilidades dentro del modelo.
