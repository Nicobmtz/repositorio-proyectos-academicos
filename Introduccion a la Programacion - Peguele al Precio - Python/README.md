# Proyecto: Juego ¡Péguele al Precio!

Este fue el primer proyecto de programación que realicé, e implicó el aprendizaje autodidacta del uso de librerías en Python.  
El juego consiste en seleccionar, de una lista mostrada en pantalla, el precio que el usuario considere que corresponde al producto mencionado.  
Dependiendo de qué tan cerca esté el usuario del precio real del producto, se suman o restan puntos.  
Al finalizar la partida, el jugador puede elegir si volver a jugar o regresar al menú y consultar el ranking.

Para este trabajo se nos brindaron las funciones principales vacías para completar, junto con un `main` básico con las configuraciones iniciales.

---

## Objetivo del proyecto

- Crear el primer programa interactivo completo, integrando estructuras básicas como bucles (`for`/`while`) y condicionales.
- Aprender a trabajar con librerías externas.

---

## Tecnologías y herramientas

- **Lenguaje:** Python  
- **Librería:** Pygame  
- **Entorno de desarrollo:** PyScripter

---

## Mi participación

- Implementé las funciones básicas del juego.  
- Integré sonidos y la pantalla de diálogo para que el usuario ingrese su nombre.  
- Programé el guardado de los datos del resultado de la partida.  
- Implementé la sensibilidad al mouse y la detección de clics sobre los precios.

## Posibles mejoras

- Mejorar la detección de clics para evitar errores al seleccionar fuera de las opciones válidas.  
- Optimizar el guardado de resultados para que se reflejen correctamente al finalizar la partida.  
- Refactorizar el código principal para:
  - Reducir el uso excesivo de `if/else`.
  - Capturar excepciones.
  - Dividir el código en funciones más pequeñas para mejorar la claridad y reducir la necesidad de comentarios explicativos.
