# Proyecto: Juego Histeria

Este proyecto consiste en un juego donde el usuario debe seleccionar una celda dentro de una cuadrícula para que cambie de color.  
El objetivo es lograr que **todas las celdas tengan un color distinto** y que **ninguna tenga vecinos del mismo color**.  
Al inicio, se presenta un menú donde el usuario puede consultar las reglas del juego y seleccionar la dificultad.

---

## Objetivo del proyecto

- Implementar una interfaz gráfica con manejo de eventos.
- Aplicar arquitecturas de interfaces de usuario.
- Utilizar el principio de *Separated Presentation* para separar lógica y vista.

---
