# Proyecto: Juego Histeria

Este proyecto consiste en un juego donde el usuario debe seleccionar un cuadrado dentro de una cuadricula para que cambie de color.
Cuando todos los cuadrados tienen un color distinto y no hay vecinos co el mismo comlor, el usuario gana.
Al inicio se le presenta al usuario un menú donde puede consultar las reglas del juego y seleccionar la dificultad.
---
##Objetivo del proyecto

- Implementar interfaz gráfica y manejo de eventos.
- Implementar arquitecturas de interfaces de usuario.
- Aplicar el principio de *Separated Presentation*
---

##Tecnologias y Herramientas
- **Lenguaje:** Java
- **Librerias:** WindowBuilder
- **Entorno de desarrollo:** Eclipse
